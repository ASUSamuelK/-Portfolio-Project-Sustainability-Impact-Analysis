# Intel Sustainability Impact Analysis

## Overview
A SQL-based data analysis project completed as part of the Global Career Accelerator portfolio program. This project analyzed Intel's enterprise device and environmental impact datasets to quantify energy savings and CO₂ reductions across thousands of devices, delivering strategic insights that informed Intel's device repurposing program.

## Tools & Technologies
- SQL (multi-table joins, GROUP BY, ORDER BY, HAVING, conditional filters)
- SQLpad
- Data Wrangling & Aggregation
- Cohort Engineering & Regional Segmentation

## Dataset
Intel enterprise device dataset containing device metadata, usage records, and environmental impact metrics segmented by device type, age, and region.

## What I Did
- Engineered multi-table SQL joins to connect device records with environmental impact data
- Designed age cohort logic to segment devices by lifecycle stage and identify repurposing opportunities
- Applied regional segmentation to compare CO₂ and energy savings across geographic markets
- Translated complex, multi-dimensional data relationships into clear, actionable business insights

## Key Results
- Quantified CO₂ and energy savings across thousands of enterprise devices
- Identified highest-impact device categories and regions for repurposing initiatives
- Delivered strategic recommendations adopted into Intel's enterprise device sustainability program
